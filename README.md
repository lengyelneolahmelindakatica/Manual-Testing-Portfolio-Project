# Manual Testing Portfolio Project

This repository presents a real-world manual testing project created during a QA training and project work.

## Scope
The project includes:
- Test case design based on functional requirements
- Execution of test scenarios
- Bug reporting in a structured format
- Testing performed in an Agile-like workflow (sprints, test cycles)

## Content
- `test-cases.md` → Selected test cases (translated to English)
- `bug-reports.md` → Documented bugs found during testing
- `images/` → Screenshots from the original Jira environment

## Notes
The original project environment was in Hungarian.  
All materials have been translated and adapted into English for demonstration purposes.

## Focus Areas
- Functional testing
- Negative testing
- UI validation
- Basic usability issues

---

This project demonstrates practical experience in:
- Writing test cases
- Identifying and documenting bugs
- Working with test management tools (Jira, Zephyr)
